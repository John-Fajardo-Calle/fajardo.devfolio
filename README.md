# 💼 fajardo.devfolio

**Sitio web profesional de portafolio de John Fajardo – Desarrollador Backend & Ingeniero Mecatrónico.**

---

## 🚀 Tech Stack

- ⚛️ React + Vite
- 🎨 TailwindCSS (modo oscuro/claro)
- 🌐 i18next (internacionalización ES/EN)
- 📦 JSON dinámico (para proyectos)
- 📄 CV visual en línea + descarga automática
- 🧠 Context API (idioma, tema)

---

## 📁 Estructura del sitio

- `/` → Home
- `/cv` → CV visual (idioma dinámico)
- `/projects` → Listado filtrable de proyectos
- `/projects/:id` → Vista individual del proyecto
- `/about` → Perfil personal y contacto

---

## 📄 Funcionalidades clave

- 🌗 Modo oscuro/claro con persistencia
- 🌐 Cambio de idioma (ES/EN) instantáneo
- 📥 Descarga automática de CV según idioma
- 🗂️ Proyectos con filtros por tecnología
- 📱 Totalmente responsive
